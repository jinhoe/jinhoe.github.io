I just created a new automation workflow that will publish my written markdown note as a post in my website.

I'm using my favourite iA Writer to pen down my thoughts via iOS or macOS. Once I'm done, I drag the file to a folder with Folder Actions configured. The rest of the tasks will be performed by script. That's it.

My personal website is hosted on [GitHub Pages](https://github.com/jinhoe/jinhoe.github.io){:target="_blank" rel="noreferrer"} for free. Isn't it amazing?
